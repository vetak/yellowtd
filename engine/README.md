# engine

В этой папке живёт игровая симуляция.

## Что здесь есть
- `path.js` — расчёт маршрута.
- `sim.js` — основной игровой цикл: движение крипов, атаки башен, урон
  (вкл. сплэш, замедление, яд, залп, цепную молнию chain/chainRange),
  апгрейд, продажа, волны и экономика.

## Что можно менять
- Баланс логики.
- Алгоритмы выбора цели.
- Поведение волн.

## Что нельзя ломать без понимания
- Детерминированность симуляции (никакого Math.random и DOM в engine).
- Разделение логики и рендера.
