# ui

Папка отвечает за интерфейс и ввод.

## Что здесь есть
- `menu.js` — меню (главное, пауза, выбор версии карты с превью маршрутов,
  сложность, настройки).
- `ui.js` — HUD, панель башен, подсказки, обработка действий.
- `storage.js` — сохранения (слот на каждую версию карты:
  `yellowtd.save.<versionId>`) и пользовательские настройки.
- `platform.js` — единственное место, знающее про Tauri (выход из приложения,
  файловые диалоги экспорта/импорта); в браузере деградирует в скачивание/загрузку.

## Что можно менять
- Кнопки.
- Подсказки.
- Панель информации.
- Управление и хоткеи.

## Что важно помнить
UI вызывает команды симуляции, но не должен дублировать боевую логику из `engine`.
