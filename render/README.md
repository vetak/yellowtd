# render

Папка отвечает за визуализацию.

## Что здесь есть
- `renderer.js` — отрисовка поля, башен, крипов, projectile-эффектов и маркеров.

## Что можно менять
- Цвета.
- Формы башен.
- Визуальные эффекты.
- Читаемость боя.

## Важный принцип
Рендер должен читать состояние, а не переписывать боевую логику.
