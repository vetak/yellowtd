# YellowTD

Браузерная tower defense в пустынном сеттинге, вдохновлённая GreenTD из Warcraft 3.

**Играть онлайн:** https://vetak.github.io/yellowtd/

Версия **1.1.0**: «Темп волн» — следующую волну можно отправить досрочно,
поверх текущей, за убывающий бонус золота (до двух волн одновременно в
поле) + подсказка для новых игроков, переходы экранов, боевая обратная
связь + режим «Эндлесс» (бесконечные волны без победы, счёт = волна) и
испытания-модификаторы (без продажи, одна башня на тип, половина золота,
ускоренные крипы) + сложность «Кошмар» и таблица рекордов + звук и музыка
на WebAudio (синтез в рантайме, без файлов) + две версии карты на выбор —
- **«Классика»** — пустынный серпантин, 7 типов башен, 36 волн, три босса;
- **«Каньон»** — узкое ущелье с 14 поворотами, 6 башен (включая «Башню бури»
  с цепной молнией), 24 волны, два босса.

Спец-волны (иммунитет, реген, герои, бонусные), три сложности,
у каждой версии карты своё автосохранение, экспорт/импорт сейвов.

Управление: мышь, клавиши 1–7 — башни, пробел — пауза, Esc — меню.
Shift/Ctrl+клик — серийное строительство, Ctrl+клик по «Улучшить» — улучшить все.

Тесты (нужен Node.js): `node test/smoke.js` и `node test/dom-smoke.js`.
